# Tic Tac Toe
Two player Tic-Tac-Toe game.

# Demo
Check out the live demo of the project here https://pavithiran-saravanan.github.io/tic-tac-toe/

# Built With
- HTML
- CSS
- Javascript

# Features
- Change player name
- Player turn indicator
- Marker placement guide
- Highlight winning cells
